# **App Name**: Arcay3Dlabs Landing

## Core Features:

- Product Showcase: Display pre-printed 3D models available for purchase in an engaging grid layout.
- Custom Order Form: Allow users to submit custom 3D printing requests with file uploads and detailed descriptions.
- 3D Model Upload: Enable users to upload .stl or .obj files for custom print quotes.
- Material Selection: Allow users to specify desired materials like PLA, ABS, or resin for their custom orders.
- Gallery: Showcase high-quality images of completed 3D printing projects.

## Style Guidelines:

- Primary color: Electric blue (#007BFF) for CTAs and key elements to draw attention.
- Secondary color: Dark gray (#343A40) for main text and a light gray (#F8F9FA) for section backgrounds to create contrast.
- Background color: Pure white (#FFFFFF) for a clean and spacious feel.
- Headline font: 'Poppins', a modern sans-serif with a geometric touch, to convey precision.
- Body font: 'Roboto', a legible and clean sans-serif font, for easy readability.
- Simple, modern icons to illustrate the service process.
- A clean, minimal layout that reflects precision and technological sophistication.
